Project Brief

Cooney & Co. needed a polished and cohesive website to represent their event-planning business. While the final site was built in Wix at the client's request, my primary contribution centered on UI design, branding, visual hierarchy, and accessibility-driven interface decisions.

A major part of this project involved crafting a comprehensive brand board that would define the visual direction of the entire site — including typography, color palette, mood inspiration, and overall aesthetic tone. This brand foundation guided the full UI system across desktop and mobile layouts.

Design Objective

Create a clear, elegant, and emotionally resonant interface that communicates the sophistication of Cooney & Co. while ensuring:

  • Strong visual hierarchy
  • Consistent brand identity
  • Accessible color contrast and typography
  • A modern, inviting experience suitable for weddings, milestones, and professional events
  • A platform that could serve both individual users and enterprise clients

Our primary users ranged from technical data analysts requiring advanced features to business professionals needing quick insights from visual data. We conducted extensive user research including interviews, surveys, and usability testing to understand their needs, pain points, and workflows.

Approach & Solutions

Design Process

  • User Research: Conducted 20+ interviews and surveys with potential users
  • Wireframing: Created low-fidelity wireframes to test interaction concepts
  • Prototyping: Built interactive prototypes for user testing and iteration
  • Accessibility Testing: Worked with accessibility experts throughout development

Technical Implementation

  • React Framework: Component-based architecture for maintainability
  • Three.js Integration: Custom WebGL rendering pipeline for optimal performance
  • Progressive Enhancement: Fallbacks for browsers without WebGL support
  • State Management: Redux for predictable data flow and time-travel debugging
  • API Design: RESTful API with real-time WebSocket updates

The application launched successfully and exceeded client expectations. User feedback highlighted the intuitive interface and powerful visualization capabilities. The platform processed over 1 million visualizations in its first three months and won recognition at the WebGL Innovation Awards.

Key Takeaways

Accessibility is Non-Negotiable: Building accessibility in from the start saved months of refactoring and resulted in a better product for all users.

Performance Matters: Users notice even small delays. Investing in optimization early paid dividends in user satisfaction and retention.

User Testing Reveals Truth: Our assumptions about user needs were often wrong. Regular testing kept us aligned with actual user requirements.
