Project Overview

Cooney & Co. is a boutique event planning company specializing in weddings, corporate events, celebrations of life, and milestone gatherings. The client needed a polished online presence that clearly communicates their services and expertise while maintaining an elegant, trustworthy look and feel.

This project showcases my approach to user-centered design, from research and wireframing through high-fidelity prototypes and final implementation.

The Challenge

To design a user-friendly, elegant website that:

  • Communicate value quickly
  • Guide users through services efficiently
  • Consistent design
  • Encourages inquiries or bookings

UI Design Approach

1. Elegant Visual Hierarchy

Using typography weight, spacing, and color pairings, I created UI sections that feel airy, organized, and intentionally layered. The hero area features a bold header paired with calming neutrals, guiding the user's eye through the page effortlessly.

2. Consistent Layout System

I built a modular-inpired system with clearly defined content blocks, repeated spacing patters, and uniform style.

3. Accessible Contrast & Readability

While maintaining brand softness, I accounted for readability with larger font sizes and color contrasts.

Accessibility Focused

Accessibility was a core principle, not an afterthought:

  • WCAG AAA Compliance: All text meets the highest contrast standards
  • Keyboard Navigation: Every feature accessible without a mouse
  • Screen Reader Support: Comprehensive ARIA labels and semantic HTML
  • Motion Preferences: Respects prefers-reduced-motion settings
  • Focus Indicators: Clear visual feedback for keyboard users
  • Alternative Text: Descriptive labels for all visual elements

Reflections and Next Steps

  • Expand accessible color options for users with visual impairments
  • Use semantic HTML structure to improve SEO and screen-reader navigation
  • Revamp gallery pages
